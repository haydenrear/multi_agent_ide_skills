# Multi-Agent IDE Skills

This repository contains all the skills for the multi_agent_ide system.

## Skills Included

- **multi_agent_ide_api** - Swagger/OpenAPI documentation and API interactions
- **multi_agent_ide_controller** - Controller loop workflows and reference materials
- **multi_agent_ide_debug** - Debugging utilities and exception knowledge base
- **multi_agent_ide_deploy** - Deployment and bootstrapping scripts
- **multi_agent_ide_ui_test** - UI testing and state snapshot documentation
- **multi_agent_ide_validate_schema** - Schema validation utilities

Each skill is self-contained with its own SKILL.md, references/, executables/, and workflows/ directories.

For skill usage, see the individual SKILL.md files in each skill directory.
